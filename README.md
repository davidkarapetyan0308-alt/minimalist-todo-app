
# 📝 Minimalist Todo App

Простое и легкое веб-приложение для управления списком задач. Написано на чистом **PHP** с использованием базы данных **MySQL**.

## ✨ Основные функции
* **Добавление задач**: Мгновенное создание новых дел.
* **Приоритеты**: Возможность задать важность (Low, Mid, High).
* **Удаление**: Очистка списка от ненужных задач.
* **Статус выполнения**: Отметка завершенных задач.

## 🛠 Технологии
- **Backend**: PHP 8
- **Database**: MySQL
- **Frontend**: HTML5, CSS3
