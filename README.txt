# Simple To-Do App (PHP Sessions)

Это минималистичное приложение для управления задачами, написанное на **PHP 8**.

### Реализованный функционал:
*   **Добавление задач** с выбором приоритета (Low, Mid, High).
*   **Валидация**: заголовок должен быть не менее 3 символов.
*   **Статусы**: возможность пометить задачу как выполненную (зачеркивание).
*   **Удаление**: удаление задач из списка.
*   **Поиск**: фильтрация списка по ключевым словам.
*   **Счетчики**: отображение общего количества задач и выполненных.

### Технологии:
*   PHP (Sessions для хранения данных)
*   HTML5 / CSS3 (Минималистичный ЧБ дизайн)
*   PRG Pattern (Post-Redirect-Get) для защиты от дублей при обновлении.
